披露宴終了後友人と

いままでの撮影画像の中から掲載ｏｋの物をアップしました。

お友達とロビーで、撮影会。まだまだ、時間に余裕がある方は、沢山撮影しましょう。式場のスタッフの方と、記念撮影も。カメラマンの撮影枚数が増え ても、追加料金は、発生しないから、どんどん撮影しましょう。

ハニーズガーデン　お世話になった会場の人とも撮影。いままで、あんなに打ち合わせをしてきたのも今日の 日の為。これからは、新郎新婦お二人の生活がスタート。
山手ヨットクラブ　仲の良いお友達とのツーショット。披露宴に参加できなかったお友達とも、このタイミン グなら撮影できます。
モントレ横浜　Imamura　披露宴が終わった後、披露宴会場に入れなかったご友人の方々やご親戚の方々 との写真撮影。披露宴終了後なら和気あいあいとした雰囲気の中、撮影できます。
橙家　集合写真程は、かたくるしくないものの、仲の良いお友達が集まっての撮影。みんな帰らないように ね。
