平井コーチ披露宴！

6月14日 にあの北島康介選手のコーチの平井伯昌さんといつもトライアスロンを指導させていただいているトライアスロンボーイズ＆ガールズの亮子さんがリッツカール トンで結婚式を挙げ、披露宴に出席させていただきました。

心よりお祝いを申し上げるとともに。素晴らしい披露宴に参加させていただいたことに感謝いたします。ありがとうございました。
