遠　　足

ゴールデンウィークも終わり、中間考査までのひとときを校外学習（遠足）に 出かけました。

1学年
　日高高校生となって初めての遠足です。「さがみ湖ピクニックランド」に行きました。組紐作りをしたり、バーベキューで盛り上がりました。観覧車やジェッ トコースターにも乗り、友達との仲も一段と深まりました。
2学年
　東京見学に行きました。羽田空港に集合の後、お台場、浅草、渋谷に分かれて判別行動でした。お台場では日本科学未来館に行き、先端の科学技術を見学し、 宇宙食も食べました。これで、秋の修学旅行（沖縄）の集合はバッチリ。
3学年
　高校生活最後の遠足は、東京ディズニーランドでした。友人と一緒に乗り物に乗ったり、食事をしたり、お土産を買ったり、一日があっという間に過ぎてしま いました。とても良い思い出ができました。
