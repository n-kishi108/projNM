結婚 式＆二次会

先日blogに書いた友達の結婚式に行ってきました。

結婚式は何回出席してもいいもんですね！

結婚式に出席するたびに決まってみんなの口癖は


「オレも早く結婚して〜！」です。

でも、その口癖も２，３日経てば誰も口にしないという
パターンが多いです。

リアルにちゃんと実行に移して結婚式を行なえるのは
次は誰になるんでしょうか・・・？

出席できるのを楽しみに待っています☆


では、結婚式＆二次会のご様子を！


このメンバーで集合写真いいです。





リムジンで披露宴会場まで・・・。





披露宴でも集合写真。





元・同僚が乾杯の挨拶。





共同作業。





失笑の中、最後までがんばりました！





二次会で新婦さんには内緒でサプライズケーキ☆





喜んでくれてホントに良かった！





こんな感じのいい結婚式でした。

乾杯の挨拶、友人代表の挨拶、余興、二次会の幹事
そして、出席されたみなさんお疲れさまでした〜！
