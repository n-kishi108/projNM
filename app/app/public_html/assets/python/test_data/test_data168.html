沖縄学生が　KOBEで　向き会った　ヒト・モノ

沖縄から「防災・社会貢献ディベート」に挑戦した７人。
滞在中、さまざまな“KOBE”と向き合った。



「人と防災未来センター」の空間
　モノ。記録。記憶。文字。圧倒的に語りかけてくる。語り部のガイドさんは、あの日のことだけではなく、まちの「今」も語ってくれた。ガイドという職業ではなく、市民として対話する。僕の経験談と書物の文字だけで「自助・共助・公助」をイメージするしかなかったメンバーが、ディベート前日にしてようやく実像を結んだ時だったかもしれない。館内で懸命にメモを取るアザマキユウたちの姿に、神聖さを感じた。いつ知ったのか、ガイドさんたちが「あしたのディベート大会、がんばって！ここでも放映するよ」とエールを送ってくれた。


伊倉久恵さん（故人）
　神戸に来て、最初に会いに行ったのは、亡き人。JR鷹取駅近くで眠っている。
　毎日新聞社で唯一の震災犠牲者。近くに住んでいた。高校は隣同志だった。結婚直後で、お腹に子どもがいた。住んでいた木造アパート１階は、押しつぶされていた。名字が変わった直後の死で、位牌は新姓だった。両親は、新聞に載る名前も慰霊碑に残る名前も旧姓の「伊倉」でと願ったが、戸籍が変えられるわけもなく、かなわなかった。「婚姻届がもう少し遅かったら」と悔やんだ。彼女の夫はかろうじて生き残り、その後再婚した。
　あの日、自分は隣の家の人を１人救出し、現場写真を撮った瞬間から無気力状態に陥った。直後の救援活動も新聞社の仕事も、すべて中途半端になった。給水車に水をもらいに行ったことくらいしか思い出せない。彼女の死を知ったのは、しばらくたってから。時が経つにつれ、大阪の会社は神戸で死んだ彼女のことなど忘れていくようだった。大阪にいると「いつまでも震災震災いうな」と言っているように聞こえる。親戚でもないのに、彼女の死を日に日に重く感じた。
　放心状態から戻り、何かしなければ心がつぶれそうで、休日はすべてボランティアで避難所や仮設住宅をまわった。心は満たされず、もがいているだけだった。ただ、彼女の死を行動の原点にした。昨年、16年目にしてようやく震災にきちんと向き合う機会を沖縄の学生たちに作ってもらった。
　神戸入りした日、奇しくも大阪毎日新聞の会社説明会を終えて駆け付けたナオコが、ユリの花を買ってきてくれた。８人で供え、手を合わせた。ナオコは横浜出身だが、阪神淡路大震災がきっかけで新聞社を目指す。伊倉久恵が同じ年で亡くなったことで、深い思いを抱いてくれた。学生たちは沖縄で災害福祉の先頭に立つため神戸に来たこと、明日ディベート大会に出場することを報告した。彼女が守ってくれる気がしたが、果たして翌日のディベートで学生たちは、「防災プロのゼミ」相手に一歩も引かない、これまでで最高のパフォーマンスを見せた。



「FMわぃわぃ」代表・日比野純一さん
「多文化プロキューブ」代表・吉富志津代さん
　なぜかこの人も毎日新聞記者だった。
　1995年１月、日比野さんは毎日新聞を辞職。戦争状態だったユーゴに行くことになっていた。数日後、阪神淡路大震災が起き、ユーゴより神戸を取った。ユーゴに行っている場合ではなくなった、といえるかもしれない。鷹取カトリック教会のネットワークで、住民もボランティアもさまざまな人が集まってきた。情報のない中で孤立する多国籍の外国人たち。言葉が通じないベトナム系住民を中心に支援をした。志津代さんはスペイン語の必要なラテン系住民のために、鷹取カトリック教会のボランティア基地に関わっていた。
　在日外国人にとって、関東大震災の時のようなデマが怖い。とんでもない方向に動くかもしれない。心配した人たちが、情報を流してほしいと頼ってきた。外国人たちは、罹災証明が発行されることも、必要であることも、知らなかった。バイクに乗せて、ラジオ機材をもらってきた。この教会で始めよう。ライセンスのいらないミニFMから始まって、震災１年の1996年１月17日、多文化・多民族共生のためのコミュニティFMわぃわぃを設立した。
　現在、10カ国の言語で流す。「ラジオは伝える道具であって、目的ではない。伝えることで、人々は問題解決に動く。解決のプロセスで、別のさまざまな課題も出てくる。それを解決する仲間も必要となってくる。いかにラジオを使ってもらうか。ラジオは手段です」。震災の時に課題が掘り起こされたのだが、実は普段から取り組んでおかねばならないことだった。
　震災時、鷹取カトリック教会の神父さんは、８年もやってきてどこに誰が住んでいて、どこに埋もれているのかまったくわからなかった。ミサで使う、焼けてしまった道具を掘り出すしかなかった。それは想像を絶する悔しさだったに違いないという。「ここを救援、復興の拠点にする。共助のコミュニティを作る。復興が終わるまで、教会を建てない」と言ったそうだ。
　「震災復興のシンボルのように言われることもあるが、ラジオがやったのではない。それを忘れてしまったり、地域のためにやっていると勘違いしたら、嫌われる。組織を守ろうとすると、ミッションが狂う。ラジオがあるのは、たかとりコミュニティセンターがあるからこそ」。



防災・社会貢献ディベート大会に参加した全国の学生たち
　神戸学院大学、東北福祉大学、工学院大学、関西大学、東亜大学、京都産業大学、同志社女子大学・・・。相手は防災のエキスパートのもとで研究をすすめる「防災プロ」のゼミ生たち。かたや、非常勤講師で震災経験者の経験と語りだけで実践を繰り返してきた雑草集団。驚くべき健闘だったと同時に、プロ集団からさまざまな学びを得たのも事実。



「NPO法人ふたば」事務局長・向恵子さん
　結婚で神戸に来て29年のうちなーんちゅ神戸人。震災時、北区の自宅はそれほど被害がなかった。「同じ神戸市民として共存すべきだ」と娘に救援物資を持たせ、六甲山を超えて一緒に届けた。職場は長田区。戦災と震災を乗り越えた、昭和初期の美しいコンクリート造りの二葉小学校が近くにある。2006年に学校統廃合で廃校になり、マンション用地として売却がほぼ決まった。「震災時に1200人の地域住民が避難した、深い思いのある場所でもある。なんとか残してほしい」。地域の25団体が集まって検討委員会を結成。連合婦人会の事務局長で新長田勤労市民センター勤務だった向さんが裏方として関わった。長田区の婦人会は6000人。検討委員会だよりを毎月5000部刷って、婦人会が配布した。
　小学校で開いた「まちの文化祭」には、1000人が集まった。おばあちゃんが「この階段を登ったもんや」と杖をつきながら一歩一歩階段を上がった。地域の声を受け、いつしか市は「地域で運用するなら・・・」。やがて奇跡が起こり、昨年11月に神戸市立地域人材支援センターとしてスタート。地域住民が役員を努めるNPO法人ふたばと、管理を行う民間企業のJVを組んで運営する。地元のキーパーソンが役員を占めるNPO法人で地域施設を運営するのは、全国でも極めて珍しい。理事会の役員は高齢者。地元のさまざまな利害関係者をまとめていけるかどうかがキモだった。外部の人でしかできない役割がある。向さんは事務局長として、理事会の一歩先を見ながら「提案はするけど決めるのはみなさんですよ」の姿勢を貫いた。地域の声を吸い上げるには「地域の実力者を手間ひまかけてコーディネートできる事務局が必要。私の後継者も育てないとダメ。次の代も外部の人です」。
　「小学校が残ったんは、地域のみんなの成果だということを忘れたらあかん」。神戸弁で語りかける向さん。沖縄の後輩たちに優しいまなざしを向けた。こんなことがあったからがんばれた、を伝えたいという。学生たちは、この事例を那覇市立久茂地公民館の利活用検討会で報告・プレゼンする。読谷の後輩になるヒガユミエが、リーダーシップをとるかな。



「神戸新聞社デジタル情報部」次長・西栄一さん
　震災で東灘区の自宅は倒壊した。結婚したばかりの妻のおなかに、子どもがいた。まわりには特ダネになるものが山ほどある。全国紙の記者は来られないのだから。記者魂と、横で泣いている妻のどちらをとるか。妻を見た。「自分は荷物だと思われている」。顔に書いてあった。神戸に身寄りがなく、大阪の実家に瓦礫を乗り越え歩いて行って預けた。神戸新聞は社屋が倒壊した。発行すら危ぶまれる危機。当然神戸を離れることができず、別居を余儀なくされた。１年ほどたち、実家に行くと子どもが生まれていた。妻は、ひどくけなした。当時のことになると、今も言葉が返せない。
　大阪から駆け付けた仕事場では、上司は「おまえは会社にすぐ来ると思っていた」。その言葉は、今も心の中にべったりと残っている。記者としても、夫としても、中途半端な1995.1.17だった。震災後、７～８年は熱病的に「何かせなあかん」という思いにかられた。休みは全部ボランティア活動にあてた。仮設住宅などまわった。震災を避けて通れなくなった。それまで考えていた、教育や障害についてライフワークにしようと思っていた記者人生のプランは吹き飛んだ。人生も会社でのポジションも大きく変えた。「いかに伝えるか」、しかなくなった。
　アウシュビッツ、沖縄、水俣。記憶の継承をどうするか、取材してまわった。アウシュビッツの資料館で日本人唯一の公認ガイドをしている中谷さんも、神戸出身。「体験者とは同じではないかもしれないが、伝えられる」と聞き、希望を感じた。「体験しているからえらい」「体験していないと伝えられない」のではないんだ。ひめゆり平和祈念資料館の生き証人たちも、中谷さんを訪れたという。「私たちがいなくなっても、体験なくても伝えられる」。
　神戸市民には、震災体験者とそうでない人で深い溝がある。しかし、1995.1.17のことはみんなが知っている。神戸にいてもいなくても、テレビという共通のツールで見ているのだから、共有できるはず。伝えられる。（談）



「神戸市中央福祉事務所」課長・谷家誠司さん
　当時の仕事場は、灘福祉事務所。震災当日、半分しか職員がいない。助けを求めてやってくるたくさんの人たち。となりの病院には続々とケガ人、死者もいるだろうか。電話が鳴りっぱなし。どれも県外から「安否を知りたい」。全くわからない。情報管理は全く機能しない。遺体の名簿は手書きだ。その遺体に関わる仕事を担当した。マニュアルでは区内２カ所の寺が安置所だったが、すぐ満杯に。警察も総出で体育館や会館を探してお願いしまわった。安置所に役所の職員は 24時間張りつきとなった。交替要員なし。
　灘区の遺体は794体。家族も亡くなって看る人がいないことも。棺も何もない。館内の葬祭業者は全滅。他府県から棺桶のキットが送られてきた。徹夜でいくつも組み立てた。死亡診断書は出ない。遺族のやり場のない怒り。行政に向けられる。「ひつぎもないんか！」。怒号。罵声。本部長が泣きながら状況を説明して謝った。少し協力関係ができてきた。しかし、市内斎場は１日150体しか火葬できない。神戸市内の死者は4500人。朽ちていく遺体。ある婦人は「見るに忍びないから、そこで焼いてくれ」。自治体職員なのに、こんな場面で何もできないことが最も辛かった。厚生省は六甲アイランドで野焼きを指示してきた。局長は断った。「遺族は一生区切りがつけられなくなる」。京都で火葬できることに。が、バスがない。渋滞の中、何時間もかけて山科に。同行できる遺族は１人にした。婚約中の若い女性だけ、遺族ではなく婚約者をそっと乗せた。２月６日まで３週間、毎日続いた。
　少し落ち着いても、昼間は業務、夜間は避難所や仮設住宅まわり。ものすごい量の仕事。しかし、投げ出したいという気持ちは不思議と起こらなかった。使命感だけでやっていた。「死んでも仕方ない」という人もいた。実際に体を壊し亡くなった職員もいる。だが、「自分だけ楽したい」という職員は誰もいなかった。職員の心のケアができなかったことは、行政として痛恨の課題だった。
　話をしたらあかん、できないという感覚だった。自分の中で整理もつかなかった。13年目くらいから話してもよいかなという気になってきた。一昨年の NHKスペシャル出演は、普通の心理状態でいることは不可能なので断った。「まだ行政のあら探しをするのか」と言った。熱心なディレクターに説得された。 10分の放送に、３時間収録した。今は伝えることも、行政職員の仕事だと思っている。（談）

「神戸市障害福祉部」相談員・大山真弓さん
　直属の上司だった、K係長のことが浮かんだ。出産間近の奥さんと幼い長男を西区に残し、職場に泊まりがけで働きづめに働いて…震災後肺ガンになり、壮絶な闘病生活の末亡くなった。当時のことを共に話せるいちばん大切な同士がいない喪失感を、今日は感じた。
　灘区の遺体安置は７４６。東灘区はそれ以上の被害、総数は１０００を超えた。当時の東灘福祉事務所には３人の係長がいた。うち２人は現職で肺ガンで亡くなった。私や、当時の同僚も１人がガンになったので、かなり高い罹患率ということになる。
　学生さんが職員さんの心のケアはなかったんですか？と聞いてくださって・・・嬉しかった。
　課長は、思い出すのも辛い記憶を職務としていろんなところで話しておられることを知った。使命感を持ってお話されている課長さんの隣で、課長にとっての震災関連業務はまだ終わってないんやなぁ…と感じた。私はまだあんなふうには話せないし、東灘の資料もまとめていないけれど、私に出来ることでがんばろう、と思う。（mixiでのやりとりから）



「大阪・釜ヶ崎日雇い労働組合」副委員長・三浦俊一さん
　釜ヶ崎には27000人が居住しているといわれる。うち9000人が生活保護受給者。平均寿命は72歳、平均年齢57歳。98％が単身高齢者。九州・沖縄県出身者が最も多く、九州の中では福岡と沖縄が多い。沖縄出身者は以前と比べ減っている。センター（あいりん労働公共職業安定所）およびその界隈で仕事にありつける人は年間32万人。1993年は184万人だった。アルミ缶も重要な収入源。高齢化で仕事にありつけない人も増えている。一方で若い人は他の地域に流れて行っている。携帯電話があれば、現場のおやじさんにどこからでも呼び出されるし、他地域のアパートに住んでやっていけるケースが多いという。
　97％が男性。母子家庭の貧困は重大な問題だが、風俗産業に流れるという。男性の結婚歴は６割程度。子どもはいても「会わせる顔がない」。奇妙な自己責任がそういう心境にさせる。「選択肢のない自己責任は、強制だ」。強制から強制排除へ、社会の仕組みがある。その仕組みを解き明かすのが、私たち「釜日労」だという。自己責任という言葉が出てきたのは1993年。「小さな政府」「規制緩和」「市場原理」「民営化」を世界中に輸出し、アメリカ資本主義を拡大させようとするワシントンコンセンサスに呑み込まれる。会社の社会的責任を株主配当や内部留保に求めるアメリカ型の勢いが増していった。
　ここの人はみんな、寂しい。「団らんを教えてくれ」といわれる。声をかける時、必ず口にするのは「故郷に帰ってみたら？」。ふるさとには楽しかったことが必ずある。自慢話があるはず。（談）
