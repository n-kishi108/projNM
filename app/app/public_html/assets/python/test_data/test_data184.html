ヒラマ ツ&マホ・結婚式

６月１５日（日曜日）編　ラスト

１３：４０式場に到着。



久しぶりに生駒クルーの皆と再会。



ウコンの力を借りまくる。（あのー言うときますけど、僕一人でこれだけ飲んだんちゃうからね…）



披露宴開始。友人代表で伸明のスピーチ。うーーーーーん心に沁みるぜぃ。



スピーチ後はビール、ビール、ビールやったなぁ。ひゃっひゃっひゃっ。



ええ感じでエンジン掛かってきたねぇ。




２次会乾杯の挨拶担当９４。


新婦のマホちゃんがブライダルプランナーって事もあって最高な披露宴&２次会でした。



ヒラマツ&マホちゃん末永くお幸せに!!!
