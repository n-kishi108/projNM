PLAYSTATION Network新規コンテンツ制作発表会開催。独占配信映像第1弾はアニメ「亡念のザムド」

大内正宗   GameSpot Japan編集部
公開日時：2008/07/18 20:37

　ソニー・コンピュータエンタテインメントジャパン（SCEJ）は、7月18日に「PLAYSTATION Network新規コンテンツ制作発表会」を開催した。

SCEJビデオ配信事業担当プロデューサー 田井野氏

　SCEJビデオ配信事業担当プロデューサー田井野 賢氏より、まずは、PLAYSTATION Networkサービスの立ち上げから現在に至るまでの推移などが発表された。
　なお、2008年5月に日本国内で100万アカウントを突破し、現在も順調な推移をしているとのことだった。

　PLAYSTATION Storeの利用状況に関しては、2008年5月末日現在で、月間来場者数が40万ユニークアカウント。総コンテンツ数が約1500コンテンツ。そして、 累計ダウンロード数が1700万ダウンロード。

　そして、PLAYSTATION Storeを利用した新規サービスとして、「E3 2008」にて発表されたビデオ配信サービスを日本でも行なうことが発表され、このサービスにて配信される第1弾作品が「亡念のザムド」であると発表され た。

　「亡念のザムド」はPLAYSTATION Store独占配信になる完全オリジナルアニメーションで、「鋼の錬金術師」や「交響詩篇 エウレカセブン」など数多くのコンテンツを手がけたボンズの最新作。監督に宮地昌幸、音楽に大島ミチルなど実力派スタッフが加わった ノスタルジックSFアクション。
　HD（ハイ・デフィニション／高精細・高解像度）およびSD（スタンダード・デフィニション）映像と5.1ch サラウンドに対応しており、迫力の世界観が楽しめる。全26話を予定しており、各話3日間の有料レンタル方式で、PS3にて視聴可能だ。
　なお、北米ではすでに第1話が配信されており、ナンバーワンの配信実績を誇っているとのことだ。

平穏な学園生活が、何者かのバス爆破事件をきっかけに一変する

　プロモーションビデオ公開後、ボンズの南 雅彦プロデューサー。宮地昌幸監督。声を担当する、主人公・竹原アキユキ役の阿部敦さん、西村ハル役の折笠富美子さん、ナキアミ役の三瓶由布子さん、アク シバ役の小西克幸さんが登場。それぞれに、「亡念のザムド」に対する意気込みや印象などを語った。

左から、南プロデューサー、阿部さん、宮地監督、三瓶さん、折笠さん、小西さん

　PVとダイジェストを見ただけの印象で言うと、宮地監督が関わってきた作品の色が色濃く出つつ、それでいて新しい物語に仕上がっているように感じだ。放 映が楽しみだ。
