ミキティ 庄司 結婚披露宴 １１月２７日 に
グランドプリンスホテル赤坂で！

ミキティ と 庄司 が 結婚披露宴 を
グランドプリンスホテル赤坂 で行うことがわかった。

お笑いコンビ、品川 庄司 の 庄司智春 と歌手、
藤本美貴 ミキティ の 結婚披露宴 が、友引の
１１月２７日に東京・紀尾井町 グランドプリンスホテル赤坂
の大宴会場、クリスタルパレスで行われるとのこと。


ミキティ と 庄司 は、すでに 結婚披露宴 の招待状を作成。
４００人を超えるタレントや芸能関係者が ミキティ と 庄司 の
結婚披露宴 に出席することになりそうとのこと。

ミキティ と 庄司 は、７月３日に米 ハワイ 島ホクリアの
ガーデンで 挙式 し、帰国後の同１１日に 入籍 。

結婚披露宴 は、未定となっていましたが、１１月２７日に、
ようやく日本でのお披露目が叶うことになったそうです。

トップアイドルとお笑い芸人の 結婚 を祝う華燭の典だけに、
豪華メンバーがそろうことは確実。 ミキティ が０７年まで
在籍していたモーニング娘。のメンバーやプロデューサー
つんく♂はもちろんのこと、 庄司 の相方、品川祐ら
吉本芸人が一同に会する 結婚披露宴 は、歌あり笑いありの
超豪華エンターテインメントとなりそう♪

ミキティ と 庄司 の 結婚披露宴 、誰が司会を務めるかにも
注目が集まります。

ミキティ と 庄司 のラブラブ 結婚披露宴 が見れそうですね(^^)

良かったら、 して下さいね。
