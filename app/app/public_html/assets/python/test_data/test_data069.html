08年11月26日
京都ツアーへ行って参りました！！　part　２

さてさて後半のツアーは

今宮神社からスタート！！

大徳寺大慈院内の泉仙にて精進鉄鉢料理で満腹の

御一行様は続いて１０分ほど歩いて大徳寺の裏にある

今宮神社に向かいます！

今宮神社を参拝後

精進料理を食べたところですが、あぶりもちを

食べて頂きます！！

参拝後皆で集合写真！！

修学旅行気分です！！長井チーフにカメラマンになっていただいたので

４４名で写っています。

長井チーフありがとうございます。

今宮神社は人が少ないです。

 

今宮神社は玉の輿守りというのが売っています！！

なんとなんと！玉の輿に乗れるというお守りです！！

私の知る限りではここにしか売っていません★

私は買いそびれましたが、ほんとに欲しくなったらダッシュで買いに行きます！！

 

その後あぶりもち！

また、食べます！！

しかし、さすがに４５人分のおもちは時間がかかってしまいました。。。。

すみません。。。

しかし、やっぱりやさしいお客様笑顔で食べて下さいました。

かざりやの店員さんが１つ１つ手作り＆手焼きなもんで。。。

 

 

 待ってる間に記念写真タイム！！

食べ終わった後も記念写真！！

 

そ うこうしていると、

あっ 時間や！！出発しなくては！！！！

最 終目的地清水へ出発です！！

続 きはpart　３　です！！
