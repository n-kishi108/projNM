七五三のきもの


子供がきものを着る機会として、七五三がございま す。

本来、七五三とは、成長の節目に衣服を改め神社で壮健幸運を祈願する行事のことです。

三歳には男児女児、五歳には男児、七歳には女児がお参りします。
三歳女児には四ツ身裁ちのお着物ですが、まだ体が小さいため、帯を結ばず綿入れ被布を着用させることが多くなっております。
五歳男児には紋付羽織袴が一般的で、紋も染め抜き五つ紋など、第一礼装となります。
七歳女児は四ツ身裁ちの着物ですが、帯は格のある袋帯を飾り結びにし、帯〆、帯揚、しごきを着用し胸元には箱迫を持たせます。

当店では、お仕立てでかわいらしさを演出した和裁士が創る祝着をご提案したします。
派手さは無いかもしれませんが、着物を知り尽くした和裁士だからこそ伝えられる正統派の祝着をしっかりとお仕立てさせていただいております。
　
お気軽にご来店ください。
