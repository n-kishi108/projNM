2007年11月

「上宇部サッカー遠征」

暑い夏も終わり、ようやく過ごしやすい季節になりました。 毎年この時期になると、私の息子の入っているサッカークラブ（ＳＣ）では、 山口県上宇部市に遠征に行きます。 今年は近くのＳＣの子供達と合同で参加しました。

子供達の楽しみは、試合もそうですが、何と言ってもホームステイです。 子供達だけで、地元チームのお宅にお世話になります。 親と離れて知らないお宅で一晩を過ごす事により、子供達は貴重な体験になったと思います。

私達親もうるさい子供達から解放されて、楽しい一夜を過ごす事ができました。 ちなみに、この大会の試合結果ですが…２位グループの第三位という成績で、賞状とたてを頂く事ができました。

１１月も別府遠征に行って来ます。 子供の行事で振り回される毎日ですが、今が楽しい時期と割り切って、 これからも張り切って応援に行こうと思います。
