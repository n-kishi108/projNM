『ゼブラーマン２』無事制作発表の模様！

▼白髪姿で戦闘モードの哀川翔、ゼブラーマン続編は「死ぬ気で」 （オリコンニュース）

（画像引用先：オリコンニュース）

▼哀 川翔：「ゼブラーマン」続編に「死にものぐるい」仲里依紗は「真っ黒な」スーパーアイドルに(まんたんウェブ) - 毎日ｊｐ(毎日新聞)

▼哀川翔、驚きの白髪で登場！なぜ…？そ れは死にもの狂いの決意の表れ - シネマトゥデイ

前作はあいかわらず未チェックなんですが、今回の続編は
哀川翔さん演じるゼブラーマンに加えて

仲里依紗さん演じる”ゼブラクイーン”なる新キャラが登場とか。

世界征服をたくらむゼブラクイーン役の仲は前日が誕生日で「２０歳になって早々、真っ黒な役ですけど大丈夫かな」と笑った

仲さん、初めての悪役？！

衣装もかなりセクシー系になりそうとかで、
このへんは三池監督の前作・大ヒットしたヤッターマンのドロンジョを意識しているのでしょうね(^-^)

ヤッターマン “てんこ盛りDVD”

さらなる強敵登場となると、
ヒーローも当然”パワーアップ”がお約束。

ゼブラーマンの場合は果たしてどうなるでしょ？

ヒーロー物のパターンで、

敗北→新フォームにパワーアップ！

とかぜひやってほしいもんです(^-^)
