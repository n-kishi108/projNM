T-リーグ第１節 vs 高桑印刷 試合後にファルマ集合！ 2006.5.16

ついに！！外でサッカー！の季節です☆
日曜日の昼下がり、角間の小さなグラウンドでウグイスのさえずりの中、わいわいと練習しています。
先輩の皆さん、お元気ですか？
ファルマジスタは新１年生11名を迎え、去年にもまして大所帯となりました。昨年度シーズンからさらに戦力アップです。
そして新年度。T-リーグも新チームとして高桑印刷さんを迎え、計６チームによる前期ステージが始まりました。
5月16日には初戦を白星で飾ったファルマジスタ。第２節３−０、第3節2 ー1で勝利して3連勝中☆　今期、躍進の予感。。
そして前期ステージ優勝をかけたvs白山戦、価値あるドローで乗り切り、次節vs NEC戦での優勝を目指す！
白山市体育大会では、高校生に走り負けするも、なんとか３位入賞！！（表彰状ありv）体力アップが今後の課題です。
