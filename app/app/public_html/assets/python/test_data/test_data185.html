2009年03月27日

「相模原RISEチア リーダーSuns 2009」　始動！

RISEチアリーダー榎本さんからの投稿です！

 
ファン感謝祭よりさかのぼること約１週間前、「相模原RISEチアリーダーSuns 2009」の活動がスタートしました！
 
『練習する場所がない！・・・』

今までは練習するスタジオをチームに提供して頂いていた為、2009年度の活動にあたり練習場所の確保は重要な課題のひとつでした。

週末は淵野辺グランドでの練習を予定していましたが、平日の夜定期的に練習できる場所がなかなか見つからなかったのです・・・。

そこで力を貸してくださったのが、＃80吉野選手やチアリーダー彩がインストラクターとして勤務させて頂いている東京・青山にある幼児体操教室と託児付きピラティススタジオ「ＯＰＴＩＭＩＳＴＡ」（http://www.optimista.co.jp/）のオーナーで、営業時間後、私たちの為にスタジオを貸して下さることになりました。

明るく綺麗で清潔感あふれる、アットホームな雰囲気のスタジオ壁一面鏡張り・・・

駅から非常に近いという立地条件・・・

どれをとっても私たちにもったいないほどの素晴らしい環境です。

そのような場所を練習場所として提供して下さったことに、心より感謝しております。

本当にありがとうございます！
 
この日は簡単な自己紹介や「相模原RISEチアリーダーSuns」としての心構え、目標などの説明があった後、アップを行い、早速ダンスの振り写しを3曲行いました。

初回の練習を終えて、みんなの笑顔がキラキラしていたのがとても印象的でした。

チームをよりよくして行く為に、ベテラン・ルーキー関係なく提案出来る、言い合える環境つくりをして行きたい

そして、ファンの皆様と喜びや感動を分かち合える応援、共に歩むチームつくりをして行きたいと思います。

それを実現するのは容易なことではありませんが、たくさん考え、笑い、泣き、悩んで、少しずつ私たちの考える最高のものを築き上げて行きたいと思います。
 
ルーキー3名を迎えて、総勢10名でスタートした「相模原RISEチアリーダーSuns 2009」！

この10名のメンバーで皆様とお会い出来る日を楽しみにしています！！






※写真　初回の練習を終えて「ＯＰＴＩＭＩＳＴＡ」の前にて
