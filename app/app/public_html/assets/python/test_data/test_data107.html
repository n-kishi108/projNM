キャトルセゾン
「キャトルセゾン」・・・ フランス語で「四季」を意味します 緑の木立に囲まれたオーベルジュ　キャトルセゾンは、 浜名湖の湖岸通りを少し上がった 小高い山の中腹にあります。 眼下に四季折々、様々な表情を見せる浜名湖を望み、 まさに自然の中、伸びやかな開放感に包まれたレストランです。 浜名湖や舞阪港の新鮮な魚介など、 地元の恵まれた食材を中心に、 繊細で身体に優しいフランス料理をご提供しております。 大人のレストランウェディングも☆

2007年04月29日

４ 月29日晴れ

こんばんわぁ♪

すばらしぃお天気に恵まれた今日の結婚式。
とりあえず写真です。



新緑がすばらしくキレイ！
そして木漏れ日を浴びる新郎新婦さん。
和装でした。カッコイイ！



浜名湖も素晴らしくキレイでした。



雲ひとつない青空の下、ご友人とお写真。
新婦さんほとんど見えてませんがｗ



こちらはメインテーブル
お花の間には新郎さんの力作を飾りました。
がんばりましたねぇ。



↑こちらはご友人の作品だそうです。
お名前であいうえお作文。そしてフクロウ(不苦労)



シャンパンタワーで使用した
ＢＩＧボトル(通常サイズ４本分)に暖かいメッセージを書いてもらいました。



後半の１００円ジャンケンでの勝者はなんとこのお子さん(ジャンケン強っ！)。
戦利品を手に照れながらも新郎新婦へお祝いを言いました。



新婦さんからブーケを受け取ったご友人たち。

全てのゲストの方たちと楽しい時間を過ごしたいということで中座もなしでした。
形式にこだわらずお二人らしい楽しいパーティになりました。
本当におめでとうございました！
