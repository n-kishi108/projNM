思っ た時から動き始めてる・・・

2010.03.26（金）
プランナー : 三 浦弘靖

皆さんこんにちは！

先日の結婚式で思わず号泣してしまったことが御座います。

 

実は当日は新婦様のお誕生日♪

彼女を喜ばせようと新郎様とたっくさんの打合せをさせていただきました。

キーワードは・・・・

『誕生日』

『伊良湖岬』

このキーワードで作戦を練りに練り当日。

新郎様、プランナー、司会者、キャプテンでスクラムを組み、いよいよ感動大作戦 スタート！！

披露宴も開宴となり司会者から『ケーキ入刀』のアナウンスが・・・

が、あるはずのケーキがございません。

？？？？？

新郎様、見てきてくださ～い。

・・・・・

会場が突如暗くなりお二人の思い出の曲『GReeeenのキセキ』が流れ始めた。

そしてケーキを運ぶ満面の笑みの新郎様♪

そしてこのケーキは・・・

お二人の大好きなリラックマ　タ キシード＆ドレスバージョン♪

そしてなんといってもこのオリジナルケーキは・・・プロポーズの場所『伊良湖岬』を表現！

愛を誓ったあの場所で・・・

このプルーナから始まるストーリー♪ステキです♪

そして本日は新婦様のお誕生日だったのです！

会場の皆様から♪HAPPY BIRTHDAY♪の大合唱、涙涙のケーキ入刀になりました！

 

 

 

 

 

 

 

 

 

 

斉志さん☆希さん

サプライズ大成功！！

皆様も大切な人の為に最高のサプライズを考えてみませんか？

是非是非プランナーへご相談下さいませ♪
