2008年04月13日

いちご便り!!!

毎度お馴染み大富農園です。

春の陽気に眠くなる今日この頃です。皆様どうお過ごしでしょうか？

大富農園のイチゴ狩りはまだまだ完熟したイチゴが皆様をお待ちしておりますよ☆

当園のイチゴ狩りは全面バリアフリーで予約不要!!立ったままイチゴ狩りが出来、

3種類の違った味のイチゴを食べ放題なので、ベビーカーを押しながらや車椅子でも

楽々イチゴ狩りが楽しめちゃいます☆

また今月いっぱいは当園のホームページ又はブログを見たと言って頂ければ、

中学生までのお子様の入園料１０％OFFとなっております!!!!



デート中の所を撮らせて頂きました、うーん初々しいです☆

カップルのお客様も是非是非いらして下さい!!!!!!
