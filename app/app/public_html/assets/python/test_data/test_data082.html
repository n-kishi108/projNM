ミス青山コンテ スト2008 久富慶子

2008.09.28

★PARTY★

９月24日は他大学と合同のパーティーに参加させていただきました＾＾ｗｉｔｈ広告研究会の先輩★藤田先輩★大槻先輩★見瀬先輩★

 

汐留駅で待ち合わせ♪ということで新橋～汐留まで『ゆりかもめ』で向かいました!!

 

初ゆりかもめ!!!!!乗車時間１分!!!!!笑

 

新橋から汐留って徒歩でも６分くらいだったみたいです(＾＾；)

 

でもでも初ゆりかもめ♪なんだかワクワクしました(＾＾)ｖ

なんと駅がドームのようになっているんです!!

思わずパシャリ♪

雨が降っても安心です♪

ゆりかもめはちょっこっと高い場所に線路があるので新橋や汐留の景色がとっても綺麗に見えました＾＾

 

私が乗った時はまだ明るかったのですが、日が暮れてから乗ったらとっても綺麗だと思います(＊＾＾＊)みなさんも是非ゆりかもめ乗ってみてください ☆☆

 

汐留駅で先輩方と合流しさっそく集合写真♪

 

移動してパーティー会場へ♪

 

パーティー会場は汐留の高層マンションの最上階!!２６階!!

 

どのくらい高いかというと・・・エレベーターで２６階へ上がる時耳鳴りがするくらいです!!ビックリ!!

 

夜景がとっても綺麗でした(＊＾＾＊)

東京タワー★

写真よりずっとずっと綺麗でした＾＾

東京タワーの高さを超えるものも増えてきましたが、やっぱり東京タワーに敵うものはない!!ってくらいきれいです♪

是非是非夜の東京タワー見に行ってください☆☆

 

 

はじめは少し緊張しながらも他大学のかたやパーティーを主催してくださったC－livesのみなさんと普段できないようなお話ができてタメになりま した＾＾

頑張っている人とお話しすると自分も頑張ろうといい刺激を受けます!!

 

 

 

パーティー会場ではこんなヘアコロンも頂きました★

THE　丸顔!!笑

とってもいい香り＆さらさらヘアーに簡単になれちゃう優れもの♪

ありがとうございました(＊＾＾＊)

 

また更新します♪

おやすみなさい＾＾ｖ
