秋季バレーボール大会　（女子９人制の部）
[ カテゴリ未分類 ]    

平成２１年１１月２２日（日）　　体育センターにて
第９回菰野町秋季バレーボール大会（女子９人制の部）が開催されました

今回は８チームの参加があり、
２つのリーグに分かれて、総当りのリーグ戦を行い、
それぞれのブロックの１位で決勝戦が行われました。

大会当日は非常に寒かったのですが、体育館の中は熱気あふれていました

熱戦続きの中、それぞれのリーグを制したのは・・・
　Ａブロック：オール菰野Ａ　　　Ｂブロック：ハーモ ニー　　の２チーム

　　　そして見事優勝されたのは・・・

　　　　　　　　　　　　オール 菰野Ａ

　　
　　　そして準優勝は・・・

　　　　　　　　　　　　ハーモ ニー


　　という結果になりました　おめでとうござい ます

　　大会に参加されたチームの皆さん運営のご協力本当にありがとうございました
　　そして、おつかれさまでした
