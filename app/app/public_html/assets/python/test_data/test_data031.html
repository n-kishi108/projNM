修学旅行

10月2日から3日間、2年生は、

『日本の伝統文化に触れ、豊かな生き方を創造しよう！〜伝統と文化の香 る、美しい国「日本」〜』

とテーマを設定し、大阪・京都へ修学旅行へ行ってきました！

本年度は、、人権学習、職業体験、伝統文化財見学、上方文化などたくさん 学習し、すばらしい思い出が出来た修学旅行でした。

 

　

集合写真（USJにて）

 

1組集合写真（金閣寺にて）

  

2組集合写真（渡月橋にて）

 

将来について考えた「私の仕事館」。
身近な職業や伝統的な職業を体験しました。
写真は京都独自の京象嵌を製作しているグループです。
金を打ち込むのがとても難しかったようです。

 

歴史を感じさせる「竜安寺」。
石庭は、当時、差別を受けていた人々の手によって作られました。庭師たちは、差別に負けず高度な技術を伝えました。15の石に込められた庭師の思いを感じ ました。

 

 

池に囲まれた金の城「金閣寺」。
見ているみんなの目も金色に輝きました。
金閣寺の畳は八代で作られたものが使われています。

釘を使わず建てられた「清水寺」。
昔の建築技術の高さに感動していました。

 

人権について考えた「リバティおおさか」。
差別を受けている人の資料の展示を見ました。
人を見下すことの残虐さを感じて、 自分自身を見直しました。
展示を見ているうちに感傷的になって、世界が平和になることを切に願いました。そして、これからは思いやりをもって人に接することを心に誓いました。


まるで、アメリカの町「USJ」。
日本と同じように、他の国々にも大切に受け継がれている文化を知りました。
アトラクションの説明を笑顔でわかりやすく、説明してくれたクルーの人。
ごみひとつ落ちていない広大な敷地。
すみずみまで掃除をしていたクルーの人。
ぬいぐるみをかぶっているクルーの人。
でもつらそうなそぶりをまったく見せず、かわいいしぐさでお客さんに夢を与えていました。
私たちの社会は、たくさんの人々が作り上げた伝統と文化、仕事によって支えられていることを学びました。
