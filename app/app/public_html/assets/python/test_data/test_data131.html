コンパニオンアニマル事業部では、伴侶動物の予防・治療のための動物用医薬品・フード等を提供するのみならず、動物病院の獣医師・スタッフの皆様への情報 提供を通して伴侶動物医療の向上に寄与することを目指しています。

いずれの製品も、全国の動物病院様を通じて、処方ならびにお求めいただけます。
