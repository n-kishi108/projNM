インタレストマッチ-広告掲載について

2007年09月08日

結婚式

先日、大学時代、私の部活仲間だった子（新婦）の結婚式に行ってまいりました。

昔から『こいつは、結婚一番早そうだなー』と思っておりましたが、

まんまと、先、越されてしまいました

場所はラグナです。朝十時からの割と早めの時間帯。

実は、私たち部活メンバーは彼氏さん（んっ、もう旦那さんかな？）とは面識がなく、結婚式で始めてのご対面。

付き合って一年たつかたたないかの割と早めのゴールイン。私たちをあっという間に抜き去っていきやがりました。

式はラグナの中の広場みたいなとこで行われました。
なんと、二人とも学校先生。
可愛いお客さんもたくさんお祝いに駆けつけてくれていましたよ。

牧師さんがいて教会風の結婚式、指輪交換や、誓いのキスなんかもやっちゃってました



披露宴はその後、そのままラグナの別会場で。


実は新婦さん、五人兄弟。

弟が入場のピアノの演奏、双子の妹が２人で披露宴の司会（２人とも色違いのお揃いの服でなかなかでした。）、

で、お姉ちゃんがビデオ撮影と兄弟が協力してアットホームな結婚式。





そんな中、新婦はウエディングドレスを着てとても綺麗でした。部活の時の印象と違うのでビックリでした。
（でも、ノリはいつものどうりの彼女で、そこらへんは流石だなー、と感心。）

新郎は年上で内地の方。なんでも、同じ学校で知合った社内恋愛

とても、優しくて、真面目な感じで、生徒みんなに好かれてそうでした。
（でも、少し抜けてる感じがとても良く一緒にお酒とか飲んだらとても楽しそうな方でした。）

余興はエイサーに、バンドに、ビデオ、ビリー先生と盛りだくさんでした。

僕なんかはビデオレターを流したんですけど、急遽作ったので一部、

声が聞き取れないとこがあったりしましたがなんとか字幕でカバー

で、やはり大人気のビリー先生。新郎新婦を舞台（部隊？）に呼んで、

喧嘩したときの謝りかたの運動や、嘘をうまく見破る運動などなかなか面白かったです。

でも、一番凄かったのはやはり新婦ですかね～。余興の沖縄民謡、エイサー、ブートキャンプ、

さらには自分でバンドを組んで自分で作詞した新郎への歌のプレゼントなど

まさに、この日の主人公でした。





友人代表で、２人の共通の友人が挨拶していました。かなり、いい挨拶でした。


私なんかも、前に出てちょっとした送り物と挨拶を。


テニス部のみんなです。



同期のみんなも久しぶりの集合です（内地組の何名かは来れなかったんですけどね）


みんなで１枚。






二次会は特になかったので、テニス部でカラオケに．．．。


テニス部で遊ぶのは２，３年ぶりかな？相も変わらずのみんなでした♪






いいな～、結婚！

亜香利、結婚おめでとう～♪

俺なんかも後に続くからね～！！！
