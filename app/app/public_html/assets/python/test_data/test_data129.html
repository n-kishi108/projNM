イギリス式結婚式

友人の結婚式をした場所はロンドンから車で２時間くらいのところにある
FarnhamっていうSurrey と Hampshireの中間地点にある街でした。

「Northbrook Park」という素敵な建物。

↑
この建物の中でセレモニーと食事＆食後のダンスをし、
この時は写真撮影にみんな外にでたとこ ろ。

いや、正確には
１４：３０セレモニー開始　→　テラスに出てシャンパンで乾杯　→　おしゃべり　→
写真撮影　→　着 席　→　食事　→　スピーチ　→　ダンス　→　ダンス
→　ダンス　→　ダンス　夜中まで続く　→　ホテルに戻って飲み直し
っていう。
ほ んっとこっちの結婚式って長いっっっ。

最初のセレモニーと食事している数時間以外はほぼ立ちっぱなし。
みんなほんとに体力あるよ ね。

私なんてたいてい最初のシャンパン２杯で酔い、食事しながらワインも
飲んだらもう６時くらいには眠くなってるので・・・
最 初のころはこの１０時間ぶっとおしで続くイギリス式結婚式に
慣れずにほんと疲れました。　まぁ、今でもまだ慣れたとは言い難いけど。


今 回の結婚式。
イギリスはここのところずーっとお天気が悪かったらしいんだけど
この日はカラッと晴れてすごく気持ちのいいお天気でした。

こ のカメラマン↓　　見える？

ただのおっちゃんかと思いきや実はウェディングフォトグラファー・オブ・ザ・イヤー
（年間最優秀結婚式写 真家賞）を受賞したこともあるすごい人だったらしく、

「はーい、そっちの右の人もうちょっと後ろに〜」
「真ん中の人たち、ぐぐー と前へ〜」


って、ただフレームに収めるために指示してるのかと思ったら！

出来た写真をみて納得。



ハートのかたちになってるの！


す ごい。おっちゃん、オシャレ！
さすが年間最優秀賞受賞者。

私の結婚式の集合写真って左右の人たちがフレームに入りきれてなくて
た ぶん１０人くらいカットされちゃってって・・・
はぁああ。。。ってかんじなのでこの写真ものすっごい羨ましい。

家族の集合写真は 更にきれいなハート型になってた。



結婚式の次の日はお昼の電車でロンドンに戻り、
ちょーどやっていたテムズ川 フェスティバルに。

インドみたいな人口密度なのになぜかここだと苦ではない。


フリーランニング（パルクー ル）のステージがありました。
建物をポンポン飛んで移動したり壁を走ったり・・ってＣＧ見てるみたいで
すごい身体能力。

ＭＶやＴＶＣＭで使われてるだけあって
すごいね。最近はスポンサーも付いてるんだね。


ロ ンドン最後の食事は勿論、ＰＵＢ飯。

旦那の旧友たち８人集合でパブ。

この入れ物風になってるのがヨークシャープディングだそうで、
初 めて食べたんだけど美味しいんだね。
シュークリームの皮を厚くしてモチモチにして香ばしくしたかんじ。

何人かはここだけ残してた けど。
たぶんしょっちゅう食べてるから　「あー、またこれかー」って感じなんだろうね。

そして不覚にもこのイギリス式ソーセージ を美味しく感じてしまった・・・
ベーコンも美味しく感じたし、
インドに住んでからというもの味覚が確実に変わってきている。
