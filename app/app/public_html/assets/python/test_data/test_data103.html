2009年02月20日

第１２回湯沢市 スノーバレー

時間が取れなく てなかなか載せる事が出来なかった第１２回湯沢市スノーバレー大会です！！

今年も満喫でしたよ～！



今年も安座間選手の選手宣誓で始まった第１２回湯沢市スノーバレー大会！


挨拶をする大古さんと役員の方々。毎回呼んで頂きありがとうございます。


エンジョイクラスも合わせると参加チーム多いですよ～！


ビーチバレーﾁｰﾑ！



試合風景～今年は暖かい方だったけど、やっぱり寒かったよ！


ビーチバレーﾁｰﾑのレシーブ体系！他のﾁｰﾑとは微妙に違うトコがうまくいきました！大古さんはそれを即座に見破ってました！さすが！！



スーパークラスは秋田国体６人制のメンバーも出てきてるので、レベル高いっす！



エンジョイクラス！皆楽しそうにやってました～しかしこのコートは水分が多く大変そうでした！




もりもりの雪の上でも変わらない高い打点からのショット！


俺のさーぶ。


しんやのさーぶ。


声だして盛り上がってます！



吉田兄貴のプレーシーン！


杉先生！今年もお世話になりました！


解説中の大古さんとアナウンサーの菅さん！今年も辛口トークでした！


優勝決まった瞬間！最後のマルシャットの感触まだ左手に残ってます！笑


閉会式後～スーパークラス準優勝-杉の子ﾁｰﾑ！


優勝-ビーチバレーﾁｰﾑ！






今年も沖縄では絶対味わえない、秋田県湯沢市の雪の上での バレーボールを満喫できました。ビーチバレーよりも珍プレーが続出し、楽しさ満点でした！！

今年も湯沢市バレーボール会長の高橋さん、大古 さん、吉田さん、杉先生はじめ、たくさんの方々のご尽力であの場に立てました！本当にありがとうございました。感謝してます！！
