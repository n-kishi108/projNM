仁ちゃん　結婚式

今日はAyumiのいとこの仁ちゃんの結婚式。
Ayumiだけ参列するはずが、風邪で来れなくなってしまったいとこの代わり急遽Masayaも参列する事に。

とっても幸せそうな楽しい式でした。
集合写真はこちら

5ヶ月前は自分たちが主役でしたが…月日が経つのは早いものです。

夜はあっこの大分の彼氏が実家に遊びに来ているということ会いに行ってきました。彼はとっても緊張してました。
