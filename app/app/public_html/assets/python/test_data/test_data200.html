【スナップ情報】メンエグ カップルスナップ IN 関西☆



次号の「メンズエッグ5月号（VOL.116）」（4月14日（火）発売）の
「カップル企画」で関西地方限定のカッ プルスナップを開催しちゃいますっ!!
東京から西のイケてるカップルはぜひ×2参加してちょ～だいネ☆

日程/3月15日（日）
時間/午後1時から午後4時ぐらいまで
集合場所/アメ村の三角公園

【当日、持ってきて欲しいモノ】
“ラブラブ度”が分かる「思い出の品」や「オソロの アイテム」

【以下の注意事項を必ず読んでから参加してくださいねっ！】
●4月14日（火）発売 号なので、春っぽい服装で来てください。
●雨天の場合は中止になる可能性があります。ご了承ください。
●全員掲載ではありません。また、 掲載に関してのお問い合わせはご遠慮願います。
